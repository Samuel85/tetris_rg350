# Tetris Clone for the RG350

[![Tetris clone ](https://img.youtube.com/vi/YJ5n5TuaGzM/0.jpg)](https://www.youtube.com/watch?v=YJ5n5TuaGzM "Tetris Clone")

## Built With
* SDL2
* SDL2_Mixer
* SDL2_ttf

## To compile
```
cd build
cmake ..
make 
```

* Opk file is located inside of the build directory.
